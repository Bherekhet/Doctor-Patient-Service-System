Doctor Service System

Setting up environment to run the system:
1) Download xamp from https://www.apachefriends.org/index.html and install on your pc
2) Download HeidiSQl from https://www.heidisql.com/ and install on your pc
3) Onced finished installing those two applications, run xamp and start MySql
4) Now you can start HeidiSql and run database file from main folder "Doctor-Patient-Service"
5) Finally download and install visual Studio or run the file executable file "CSC340TeamProject"

Now that you have installed and prepared your environment you are ready to test the system.
